# PostOne

> Post once. Publish everywhere.

**PostOne** は、1 回の投稿作成で複数のプラットフォーム (X、Threads、Instagram、note など) への配信・予約・結果管理を一元化できる、日本向けの SNS 投稿管理サービスです。

> [!NOTE]
> PostOne は現在開発中です。本リポジトリの内容はリリースに向けて随時更新されます。

## こんな方におすすめ

- 複数の SNS を並行運用している個人クリエイター・インフルエンサーの方
- 広報を一人で兼務している中小企業・店舗の SNS 担当者の方
- note と SNS 告知を同時運用しているライター・ブロガーの方

## 主な機能

| 機能 | 説明 |
| --- | --- |
| マルチ配信 | 1 つの投稿を複数のプラットフォームへ同時に配信 |
| 予約投稿 | 指定した日時に自動で配信 (1 分単位) |
| 下書き管理 | 下書き保存・編集・削除 |
| 配信結果の確認 | プラットフォームごとの成功 / 失敗をひと目で確認 |
| ダッシュボード | 直近の予約、最近の配信結果、接続状況、利用量を 1 画面に集約 |
