# 機能の詳細

## 投稿

### 本文

- 最大 5,000 文字です。
- 配信先を選択すると、選択した中で最も厳しいプラットフォーム別の文字数制限が適用されます。

| プラットフォーム | 文字数上限 |
| --- | --- |
| X | 280 |
| Threads | 500 |
| Instagram | 2,200 |
| note | 5,000 |

> [!NOTE]
> 対応プラットフォームは順次追加中です。最新の対応状況はリリース情報をご確認ください。

### 投稿のステータス

| ステータス | 意味 |
| --- | --- |
| 下書き | 配信していない状態。編集・削除・配信ができます |
| 予約済み | 指定日時の配信を待っている状態。配信開始前までキャンセルできます |
| 配信中 | 各配信先へ配信を実行している状態 |
| 配信済み | すべての配信先への配信に成功した状態 |
| 一部失敗 | 一部の配信先で失敗した状態。配信先ごとの失敗理由を投稿詳細で確認できます |
| 失敗 | すべての配信先で失敗した状態 |

## 予約投稿

- 日本時間 (JST) で日時を指定します。過去の日時は指定できません。
- 配信は 1 分単位で実行されます。指定時刻から実際の配信までに最大 1 分程度の遅延が生じることがあります。
- 予約のキャンセルは配信開始前まで可能です。キャンセルした投稿は下書きに戻ります。

## 下書き

- 下書きは何件でも保存できます。
- 下書きの編集では、本文・配信先・保存方法 (下書きのまま / 予約に変更) を変更できます。
- 下書きからそのまま「今すぐ投稿」もできます。

## 配信結果の確認

- 投稿詳細で、配信先ごとの結果 (待機中 / 配信中 / 配信済み / 失敗) を確認できます。
- 失敗した配信先には失敗理由が表示されます。

## 接続管理

- 配信先アカウントの接続・解除ができます。
- アクセストークンなどの認証情報は暗号化して保管されます。
- 接続を解除しても過去の投稿履歴は残ります。

## ダッシュボード

1 画面で次の情報を確認できます。

- 今月の投稿数
- 接続中アカウント数
- 予約中の投稿数
- 最近の投稿とそのステータス
