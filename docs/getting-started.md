# はじめに (Getting Started)

PostOne を使い始めてから最初の投稿を配信するまでの流れを説明します。

## 1. アカウントを登録する

1. 登録ページでメールアドレスとパスワード (8 文字以上) を入力します。
2. 確認メールが届くので、メール内のリンクを開いて登録を完了します。
3. 登録完了後、ダッシュボードに移動します。

パスワードを忘れた場合は、ログインページの「パスワードをお忘れの方」からリセットメールを送信できます。

## 2. プラットフォームを接続する

投稿を配信するには、先に配信先アカウントの接続が必要です。

1. サイドバーの「接続」を開きます。
2. 「接続を追加」からアカウントを接続します。
3. 接続済みアカウントは一覧に表示され、いつでも接続を解除できます。

> [!NOTE]
> 接続できるアカウント数はプランによって異なります ([料金プラン](plans-and-billing.md) を参照)。
> 対応プラットフォームは順次追加中です。

## 3. 投稿を作成する

サイドバーの「投稿」→「新規投稿」から作成します。

1. 本文を入力します (最大 5,000 文字)。
2. 配信先のアカウントを選択します (複数選択可)。
3. 保存方法を選びます。

| 保存方法 | 動作 |
| --- | --- |
| 下書き保存 | 配信せずに保存。あとから編集・削除・配信ができます。配信先が未定でも保存できます |
| 今すぐ投稿 | 保存と同時に、選択したすべての配信先へ配信します |
| 予約投稿 | 指定した日時 (日本時間・1 分単位) に自動で配信します |

> [!TIP]
> 配信先を複数選ぶと、その中で最も厳しい文字数制限が適用されます
> (例: X を含む場合は 280 文字以内)。詳細は [機能の詳細](features.md) を参照してください。

## 4. 配信結果を確認する

- 投稿一覧で各投稿のステータス (下書き / 予約済み / 配信中 / 配信済み / 一部失敗 / 失敗) を確認できます。
- 投稿詳細では、配信先ごとの成功 / 失敗と失敗理由が表示されます。
- ダッシュボードには今月の投稿数・接続中アカウント・予約中の件数と最近の投稿が集約されます。

## 5. 予約を変更・キャンセルする

- 予約済みの投稿は、投稿詳細から配信開始前までキャンセルできます。キャンセルすると下書きに戻ります。
- 下書きに戻した投稿は編集して再度予約できます。
